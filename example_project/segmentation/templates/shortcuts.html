General shortcuts

- move image around

- zoom with mouse scroll

Z - undo

X - redo

F - toggle polygon fill

A - enter adjust mode

D - enter draw mode

ESC - deselect any polygon and reset zoom

Draw mode shortcuts:

- add a new point

- close polygon

Q - snap to an existing line/boundary

W - snap to an existing point/corner

C - add a new point (same as left click)

Adjust mode shortcuts:

- select existing polygon or point

- add new point to existing polygon

A - select next polygon to adjust

S - zoom in on selected polygon

DELETE or BACKSPACE - delete selected polygon (if no point under cursor)

DELETE or BACKSPACE - delete selected point (if point under cursor)
